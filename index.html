빠른시작 방법

~ $ gem install jekyll bundler

~ $ jekyll new my-awesome-site

~ $ cd my-awesome-site

~/my-awesome-site $ bundle exec jekyll serve

# => 이제 브라우저로 http://localhost:4000 에 접속
